四步完成演示

1. 发起授权

客户端带上参数跳转到认证服务器。

2. 登录认证

用户输入账号密码完成身份验证。

3. 换取 Token

客户端在回调阶段用 code 换 token。

4. 访问资源

资源服务器根据角色和 scope 返回结果。
